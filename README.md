# Matreska

Matreska builds a cascaded object in which each object can be called in order. 'Rack' inspired creating Matreska.

## Installation

Add this line to your application's Gemfile:

    gem 'matreska'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install matreska
